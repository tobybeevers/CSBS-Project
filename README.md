# CSBS Project – Cyber Security Breaches Survey Analysis

## Project Overview

This project builds a harmonized longitudinal dataset from 8 waves of the UK Cyber Security Breaches Survey (CSBS), published annually by DSIT (formerly DCMS). The goal is to identify which variables are comparable across all years, prepare analysis-ready data, and support research into cybersecurity trends across businesses, charities, and education institutions from 2018 to 2025.

The CSBS is the primary UK government evidence base on the prevalence and impact of cybersecurity breaches. This project enables longitudinal analysis of that evidence — something not provided in published DSIT outputs.

## Data

- **Source:** UK Cyber Security Breaches Survey (CSBS), DSIT
- **Waves:** 2018–2025 (8 waves)
- **Format:** Tab-delimited flat files, ~300–570 variables per wave
- **Sample sizes:** ~1,900–4,000 respondents per wave

| Wave | Respondents | Variables |
|------|-------------|-----------|
| 2018 | 2,089 | 461 |
| 2019 | 2,081 | 461 |
| 2020 | 1,901 | 312 |
| 2021 | 2,285 | 420 |
| 2022 | 2,158 | 446 |
| 2023 | 3,992 | 567 |
| 2024 | 3,435 | 498 |
| 2025 | 3,836 | 527 |

Negative integers are used as missing/inapplicable codes: `-1` (not applicable), `-97` (don't know / refused), `-99` (other missing).

Supporting metadata files in `data_raw/`:
- `value_labels.xlsx` — maps numeric codes to categorical labels
- `variable_metadata.xlsx` — variable definitions and question wording

## Key Variables of Interest

| Family | Description | Waves Available |
|---|---|---|
| `samptype` | Organisation type (business / charity / education) | All |
| `sizeb` | Business size band | All |
| `sector_comb2` | Sector (businesses only) | All (coding change in 2024–2025) |
| `weight` | Survey weight for population estimates | All |
| `manage*` | Management and governance practices | All (6 stable variables) |
| `policy*` | Cybersecurity policies in place | All (7 stable variables) |
| `Step1`–`Step10`, `Any10Steps`, `Sum10Steps`, `AllEssentials` | 10 Steps / Cyber Essentials adoption | All |
| `type1`–`type12`, `typex` | Breach and attack types experienced | All (expanded in later waves) |
| `reporta`, `reportb*` | Reporting behaviour after a breach | All (21 stable variables) |
| `trained` | Cybersecurity training provided to staff | 2021–2025 only |

**Stable longitudinal core:** 172 variables are present and consistently named across all 8 waves.
